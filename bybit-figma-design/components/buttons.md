## 组件规格

### 1. Button_Primary

> 品牌橙色填充按钮，页面主操作。**无 Dark Mode 轴**（颜色恒定）。
> Set Key: `8cbf8e1a1105c7fa5533dc30b858896bb2192d1a`

- **变体维度**: Size (XL 48px / L 40px / M 36px / S 28px) × Status (Normal / Loading / Pressed / Disable)
- **总变体数**: 16

### 2. Button_Secondary (.Content Block)

> 描边/透明次级按钮。Figma 内部名 `.Content Block`。
> Set Key: `18e18e0a5ea8881a3d02fccded3fdedb680e4a49`
> **已验证 (2026-04-04)**: 所有变体 Key 有效，可正常 import。

- **变体维度**: Size (XL 48px / L 40px / M 36px / S 28px / XS 24px) × Status (Normal / Loading / Pressed / Disable) × Dark Mode (On / Off)
- **总变体数**: 38
- **注意**: 比 Primary 多 **XS 24px** 尺寸；XS 无 Loading 态

### 3. Button_Text

> 纯文字按钮，无背景无边框。用于"查看更多"、"跳过"等链接式操作。
> Set Key: `e4c620fdf2d065eaa8ea22ec9ad5b517d9a2ca21`

- **变体维度**: Size (L / M / S) × Dark Mode (On / Off) × Status (Normal / Disable)
- **总变体数**: 12
- **注意**: **无 Loading / Pressed 态**；Size 不带 px 后缀

### 4. Button_Trading

> 交易买卖按钮。Long/Buy=绿色，Short/Sell=红色。
> Set Key: `885688029963b1443fba1b395a7c0ea98e0bc390`

- **变体维度**: Type (Long/Buy / Short/Sell) × Size (XL 48px / L 40px / M 36px / S 28px) × Status × Dark Mode (On / Off)
- **总变体数**: 64

### 5. Button_Group

> 底部操作栏按钮组合，内含 Primary + Secondary 按钮。
> Set Key: `06aec95512378e841db86a6e014461d2afaa966f`

- **变体维度**: Dark Mode (Yes / No) × Button No. (1 / 2 / 3 / +Text / +Icon / +2 Icon / 2 horizontal)
- **总变体数**: 14
- **注意**: Dark Mode 属性值为 **Yes/No**（非 On/Off）

## 选型指南

- **Size**: XL=主 CTA（底部确认），L=次级，M=紧凑场景，S=行内小按钮，XS=迷你（仅次级按钮）
- **Status**: AI 生成只用 `Normal` 和 `Disable`。Pressed / Loading 是交互态，不用于静态设计稿
- **Dark Mode**: 默认 On。只有用户明确要求 Light Theme 才用 Off
