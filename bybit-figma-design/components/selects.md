## 组件规格

### 1. Select_Sort

> 排序触发器，点击展开下拉选项列表。
> Set Key: `e556715cd698b74120ef813a7dcbd918a9e68da5`

- **变体维度**: Size (S / M) × Dark Mode (On / Off)
- **总变体数**: 4

### 2. Select_Option

> 独立选项项，点击即选中，不展开下拉。
> Set Key: `81795b1f4e79bd724e8f4bbff4cca4481347d578`

- **变体维度**: Size (S / M) × Dark Mode (On / Off)
- **总变体数**: 4

### 3. Select_List/Single_Choice

> 单选列表项，带 Radio 控件。
> Set Key: `ca2aaf5c20acabd620527b7cd5eef3a68f10e711`

- **变体维度**: Type (1Line / 2Lines / 2Lines+Icon / 2Lines+Avatar / 2Lines+Currency) × Stutas (Normal / Checked / Disable) × Dark Mode (On / Off)
- **总变体数**: 30
- **注意**: Figma 属性名为 `Stutas`（拼写如此）

### 4. Select_List/Multiple_Choice

> 多选列表项，带 Checkbox 控件。
> Set Key: `e4ab75b8fb1d18608e59516c71ce77f753a4469f`

- **变体维度**: Type (1Line / 2Lines / 2Lines+Icon / 2Lines+Avatar / 2Lines+Currency) × Stutas (Normal / Checked / Disable) × Dark Mode (On / Off)
- **总变体数**: 30

### 5. Select_Card/Single_Choice

> 卡片式单选项，带选中边框。

- **总变体数**: 30

## 组合规范

### 下拉选择列表
```
Select_Sort（触发器）
  ↓ 点击展开
AutoLayout (vertical, gap=0)
  ├── Select_List item (Checked)    ← 已选中项
  ├── Select_List item (Normal)     ← 未选中项
  ├── Select_List item (Normal)
  └── Select_List item (Disable)    ← 不可选项
```

### 选项组（横排/竖排）
```
AutoLayout (horizontal/vertical, gap=8)
  ├── Select_Option (选中态用业务逻辑控制)
  ├── Select_Option
  └── Select_Option
```

### 卡片选择组
```
AutoLayout (vertical, gap=8)
  ├── Select_Card item (Checked)
  ├── Select_Card item (Normal)
  └── Select_Card item (Normal)
```

**选型指南**:
- **Sort vs Option**: Sort=展开下拉列表选择，Option=直接点击选中
- **List vs Card**: List=紧凑列表样式（适合选项多），Card=卡片样式（适合选项少、需突出展示）
- **Single vs Multiple**: 单选=Radio 控件，多选=Checkbox 控件
- **Type**: 1Line=纯文字，2Lines=双行文字，+Icon/+Avatar/+Currency=带前缀图标
- **Stutas**: Normal=未选，Checked=已选，Disable=不可选。单选列表中保持一个 Checked + 其余 Normal
